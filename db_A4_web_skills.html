CSS3 Techniques

Colours

CSS

Text colour 1: #ddd8d0 for the body text (the light warm grey used for all paragraph text)

Typography

CSS

font-weight: 400 – normal body text throughout the site

font-weight: 700 – headings, nav buttons and key labels

font-style: normal – standard paragraphs

font-style: italic – used for .italic-note and em elements

0.78rem small labels   1rem standard body   1.2rem h2 headings

Font family stack: Georgia, ‘Times New Roman’, Times, serif. Four typefaces declared.

Text Decorations & Selectors

CSS

Text decoration 1: text-decoration: underline – used on h1 headings and body links

Text decoration 2: text-decoration: none – removes underline from nav buttons and the logo link

ID selector (#hero-cta) styles the Enter button on the homepage.

Class selector (.card) styles every content panel across all four pages.

Universal selector (*) resets box sizing, margin and padding across the whole site.

Type selectors (body, h2, p, a, nav, table, footer etc.) define base styles for all standard HTML elements.
